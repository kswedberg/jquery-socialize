Overview

Quick Start Guide

Add HTML markup to your page.
<!DOCTYPE html>
<html lang="en">
<head>
  <meta http-equiv="Content-Type" content="text/html;charset=utf-8">
  <!-- more stuff goes here  -->
</head>
<body>
  <div id="social-sites">
    Lorem ipsum dolor sit amet.
  </div>
</body>
</html>
Include the jQuery core file and the Socialize plugin in the <head> of your document. After these scripts are referenced, you can reference a custom script file (preferred) to create the social bookmark links or enter the script directly in the <head> (shown below). You should also include the Socialize stylesheet (jquery.socialize.css) before the scripts. 

<link rel="stylesheet" href="jquery.socialize.css" type="text/css" />
<script src="jquery.js" type="text/javascript"></script>
<script src="jquery.socialize.js" type="text/javascript"></script>

<script type="text/javascript">
$(document).ready(function() {
  $('#social-buttons').socialize();
});
</script>

See API / Options for ways to override the default options.

Socialize Plugin API / Options

socialize(options)
Displays a list of social bookmark links appended to the matched container element(s). 
$.fn.socialize.defaults = {  // default options; override as needed
  sites: ['delicious', 'digg', 'dzone', 'reddit', 'stumbleupon'],
  containerId: 'social-icons',
  containerClass: 'group',
  linkClass: 'social-icon',
  insertion: 'appendTo',
  events: {
    /*
    click: function(event) { },
    mouseover: function(event) { }
    */
  }

};
